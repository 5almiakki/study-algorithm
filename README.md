# 알고리즘 공부 저장소

알고리즘 문제 풀이 기록을 위한 리포지토리입니다

## 📝 규칙

* 파일명: `<플랫폼>_YYYYMMDD_<문제번호>_<이름>.java`

## 🌿 브랜치 전략

* `main`: 기록용 개인 저장소이므로 항상 이 브랜치에 커밋

## 📚 커밋 메시지 컨벤션 (Conventional Commits)

```
<플랫폼> <문제 번호> <문제 이름> <풀이 방식>(으)로 폴이
```
또는
```
<플랫폼> <문제 번호> <문제 이름> <가짓수>가지 방법(으)로 폴이

- <풀이 방식>
- <풀이 방식>
...
```
